# Исследование стратегий и анализ 3-летнего backtest (2026-07-13)

## Часть 1. Что говорят данные о текущей стратегии

Текущая гипотеза (интрадей-скоринг: EMA 15m/1h + пробой свечи + объём + ATR,
фиксированный SL/TP через R/R) прогнана на 3 годах реальных свечей Binance
USDⓈ-M с комиссиями taker 0.05%, проскальзыванием 0.02% и funding.

| Метрика | BTCUSDT 15m | ETHUSDT 15m |
|---|---|---|
| Доходность за 3 года | **−89.0%** | **−84.4%** |
| Сделок | 3043 | 3029 |
| Win rate | 35.1% | 34.4% |
| Profit Factor | 0.64 | 0.69 |
| Макс. просадка | 89.0% | 84.5% |
| Комиссии | 711 USDT | 607 USDT |
| Проскальзывание | 263 USDT | 220+ USDT |
| Walk-forward (6 окон) | 6/6 убыточны | 6/6 убыточны |

Вывод однозначный и окончательный: **стратегия не имеет edge**. Убыток не
случайность — все 6 независимых полугодовых окон на обоих инструментах
отрицательны. Главный убийца — издержки: при ~1000 сделок/год комиссии и
проскальзывание составили ~97% стартового капитала. Даже при нулевых
издержках PF ≈ 0.9–1.0 — сигнал сам по себе не предсказывает цену.

## Часть 2. Что говорит многолетний опыт индустрии и науки

Собрано из академических работ и многолетних исследований (ссылки внизу):

1. **Trend following / time-series momentum — единственное семейство с
   вековым доказательством.** AQR («A Century of Evidence on Trend-Following
   Investing», Hurst–Ooi–Pedersen): стратегия прибыльна в **каждом десятилетии
   с 1880 года** на акциях, облигациях, валютах и сырье, Sharpe ~0.4 за
   декаду. Moskowitz–Ooi–Pedersen (2012): time-series momentum устойчив на
   58 фьючерсных рынках с 1985.
2. **В крипте momentum подтверждён отдельно.** Liu–Tsyvinski (Review of
   Financial Studies, 2021): сильный time-series momentum на горизонте
   **1–4 недели**; модель market+size+momentum объясняет доходности крипты.
3. **Интрадей-стратегии умирают от издержек.** Исследования intraday-алгоритмов
   крипты: наивные сигналы теряют прибыльность уже при издержках ~10 б.п.;
   прибыльность возвращает только резкое сокращение оборота (торговать только
   когда ожидаемое движение кратно превышает издержки). Подавляющее
   большинство розничных дейтрейдеров теряет деньги.
4. **Классика практики** (Turtle traders, managed futures): пробой канала
   Дончиана + трейлинг-стоп по волатильности + маленький фиксированный риск
   на сделку. Ключ — позитивная асимметрия: много мелких убытков, редкие
   большие прибыли; выживание обеспечивает риск-менеджмент, а не win rate.

Общие принципы прибыльных многолетних систем:

- горизонт от дней до недель, НЕ минуты;
- мало сделок → издержки перестают решать;
- вход по тренду, не против него;
- убытки режутся сразу (стоп по волатильности), прибыль не ограничивается
  (трейлинг вместо фиксированного TP);
- риск на сделку фиксированный и маленький;
- никакая система не прибыльна каждый месяц — оценивать можно только
  многолетнюю статистику.

## Часть 3. Новая стратегия-кандидат: «Donchian trend»

Прямое переложение пунктов выше на наш стек (реализовано, STRATEGY_MODE=donchian):

- **Сигнальный ТФ 4h, трендовый фильтр 1d** (горизонт сделки — дни/недели,
  соответствует моменту 1–4 недели из Liu–Tsyvinski).
- **Вход**: закрытие 4h-свечи выше максимума предыдущих 48 свечей (8 дней)
  И дневная цена выше EMA200 → LONG (SHORT зеркально).
- **Выход**: только трейлинг-стоп Chandelier = экстремум цены ± 3·ATR(14).
  Фиксированного TP нет — прибыль не ограничивается.
- **Риск**: 0.25% на сделку, начальный стоп 3·ATR, плечо ≤3x isolated,
  лимиты (1 позиция, 3 сделки/день, дневной стоп) сохранены.
- Объёмный и скоринговый фильтры удалены: на этом горизонте они не имеют
  доказательной базы.

Ожидаемый профиль: ~20–60 сделок/год, win rate 30–45%, редкие большие
прибыльные сделки. Результаты 3-летнего теста и walk-forward — в Части 4.

## Часть 4. Результаты нового кандидата (3 года, после всех издержек)

| Метрика | BTCUSDT 4h/1d | ETHUSDT 4h/1d |
|---|---|---|
| Доходность за 3 года | **+4.80%** | **+3.63%** |
| Сделок | 64 | 63 |
| Win rate | 37.5% | 36.5% |
| Profit Factor | **1.70** | **1.56** |
| Средний R на сделку | +0.30 | +0.23 |
| Макс. просадка | 1.53% | 2.43% |
| Макс. серия убытков | 6 | 9 |
| Комиссии + slippage за 3 года | ~6 USDT | ~4 USDT |
| Walk-forward: окон в плюсе | 5/6 (худшее −0.43%) | 5/6 (худшее −0.29%) |

Сравнение с текущей стратегией на тех же данных: **−89% → +4.8%** (BTC).
Оборот упал в ~50 раз (3043 → 64 сделки), и издержки перестали иметь значение
(711 USDT → 6 USDT). Ожидание на сделку положительное после издержек, знак
устойчив по 11 из 12 независимых окон и на обоих инструментах.

Почему абсолютная доходность скромная: риск 0.25% на сделку × ~21 сделка/год
× +0.3R ≈ +1.6%/год. Это следствие консервативного риска, а не слабости
сигнала: доходность и просадка масштабируются риском практически линейно
(при 1% риска ≈ +6–7%/год при просадке ~6–10%). Повышать риск имеет смысл
только после подтверждения статистики на Testnet.

**Статистическая честность:** 64 сделки — маленькая выборка. Грубая оценка
t-статистики суммарного R (~19R при σ≈1.5R/сделку) даёт t ≈ 1.6 — это
обнадёживающе, но НЕ статистически значимо на уровне 95%. Согласованность
двух коррелированных инструментов не удваивает выборку. Гипотеза достойна
Testnet-проверки, но не «доказанно прибыльна».

Команда воспроизведения:

```bash
python -m src.run_backtest --symbol BTCUSDT --days 1095 \
  --strategy donchian --signal-interval 4h --trend-interval 1d --walk-forward 6
```

## Честные оговорки

- Backtest ≠ будущее. Даже вековые стратегии имеют многолетние просадки.
- 3 года крипто-истории — это один-два рыночных режима, не «долголетний опыт».
  Долголетний опыт здесь — доказательная база самого семейства стратегий.
- Никакая стратегия не «будет прибыльной» гарантированно. Корректная цель:
  положительное ожидание после издержек, подтверждённое out-of-sample, и
  выживаемость при серии убытков.
- Следующий шаг всегда: 100–200 сделок на Testnet до любых разговоров о
  реальных деньгах.
