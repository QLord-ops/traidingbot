# Binance Intraday Bot v0.3 — веб-панель

## Что это

Локальный веб-интерфейс для настройки и тестирования стратегии. Панель открывается в браузере, но работает на вашем компьютере/VPS. Версия 0.3 не отправляет реальные ордера.

## Возможности

- изменение параметров EMA, ATR, объёма, риска, плеча и Reward/Risk;
- выбор USDT-инструмента;
- запуск backtest по публичным свечам Binance;
- учёт комиссии и проскальзывания;
- показатели прибыли, win rate, Profit Factor и просадки;
- просмотр последних сделок;
- экспорт CSV и JSON.

## Запуск

```bash
python -m venv .venv
# Windows:
.venv\Scripts\activate
# macOS/Linux:
# source .venv/bin/activate
pip install -r requirements.txt
python run_web.py
```

Открыть в браузере:

```text
http://127.0.0.1:8000
```

## Безопасность

API Secret нельзя вводить в обычную HTML-форму или сохранять в браузере. В будущей версии он будет храниться только в `.env` на сервере. Реальный режим должен иметь отдельное подтверждение, IP whitelist и API-ключ без разрешения на вывод средств.
